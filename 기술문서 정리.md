# HollowKnight
Imitation HollowKnight Game


# 기술 문서 정리

 // 누가, 왜, 무엇을 위해서 보는것인가
 // 개발 팀원이 게임의 이해도를 높이기 위해서 개발을 해야하는 덕분에

 - 목차
  1. 게임 소개
     - 각 지역을 탐험하며 스펙을 올리고 각 보스를 물리치는 게임

     - 《할로 나이트》에서 플레이어는 몰락한 왕국, 점점 오염되어 가는 신성둥지에 떨어진 주인공을 통해 이 고대의 왕국에 숨겨진 비밀을 찾고 자신에게 주어진 숙명을 완수해야 한다. 그 과정에서 플레이어는 신성둥지에서 살아가는 여러 벌레들과 생명체들을 만나며 수많은 적과 개성있는 보스들, 그리고 새로운 능력들을 획득해 나가며 조금씩 이 고대의 왕국의 진실에 다가가게 된다.

  2. 플로우 차트, 클래스 다이어그램

  3. 개발 방식
     - 목적, 구현 범위(불륨)
     - 하나의 씬에서 모든 오브젝트를 관리하며 플레이어는 하나의 씬에서 모든 동작을 수행합니다.

  4. 오브젝트 작동 원리
     - 구현 요소
     - 각 오브젝트의 설명
     
  5. 개발일지
	 - 어떤 자료구조를 왜 사용했는지
	 - 내가 구현하던 도중 경험한 이슈 또는 어떤 어려운 점이 있었는지
	 - 이후 이슈를 어떻게 대응했는지
	 - 왜 그렇게 대응했는지

  6. 개발 환경
     - 소스코드 링크[깃]
     - 동영상링크[유튜브]
     - 편집툴 -> Vs, 포토샵, ppt툴, 영상편집툴, 일정관리툴
